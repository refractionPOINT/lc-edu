LimaCharlie MarketPlace

This course is an introduction to the LimaCharlie Add-ons Marketplace.

The Add-ons Marketplace enables LimaCharlie users to subscribe to various APIs, Lookups and Services at the click of a button.

In this course, LimaCharlie founder, Maxime Lamothe-Brassard will cover the following.

  1. What is the LimaCharlie marketplace and what does it look like: a high level tour.
  2. Exploring Services
  3. Exploring Lookups
  4. An example of using the VirusTotal API integration to examine file hashes via detection rules.
  5. An example of using the IP Geo API to write detection rules that consider the endpoints geolocation.
  6. How to create your own private or public Add-on for fun or profit.
