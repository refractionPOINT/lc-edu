virustotal

VirusTotal is an online platform used to share potential malware samples and the related scan results from a multitude of Anti-Virus engines.

Setup

Set the VirusTotal API Key in the Integrations view within your organization, using the key provided by your VirusTotal account. Subscribe to this vt API add-on.

Usage

When enabled, you can use the VirusTotal API in the detection part of a D&R rule to match based on the number of Anti-Virus engines that have flagged a file hash as malicious.
