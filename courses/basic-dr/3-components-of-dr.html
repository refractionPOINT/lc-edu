Once installed the LimaCharlie Agent is able to track over 70 Event types as they take place on the endpoint. These events produce telemetry which is shared with the LimaCharlie Cloud in real-time. The telemetry is evaluated against a collection of Detection & Response (DR) rules in the LimaCharlie Cloud. DR rules are generally applied on a per-event basis. When the rule is applied the detection component of the rule is processed to determine if it matches. If it matches the response component is actioned. Response actions can include tasking the sensor, isolating the host from the network, sending an alert, along with many other possibilities.
