This is an introductory course for the LimaCharlie Security Infrastructure as a Service (SIaaS) platform. This course is designed to give the user a strong foundation from which to start leveraging the full power of LimaCharlie.

The course starts with a survey of the major components and concepts of LimaCharlie's Security Infrastructure as a Service. From this broad overview we zoom into the details behind the Detection & Response engine, followed by a web application walk through 

At the end of this course the user will:

  • Have a clear understanding of what LimaCharlie is and the scope of its capabilities.
  • Understand the principles and mechanisms that make up the LimaCharlie Detection & Response engine.
  • Be able to create simple Detection & Response rules, outputs and alerts.
  • Navigate the web application.
