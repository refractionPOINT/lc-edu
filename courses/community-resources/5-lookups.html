Lookups

Creating a lookup add-on enables you to create a list that you can use as part of D&R rules. Once in place, you can refer to it using the op: lookup D&R rule with a reference to your add-on looking like resource: lcr://lookup/my-lookup-name. Lookups support a few structures.
  • Newline-separated values.
  • JSON dictionary where keys are the elements of the lookup and the values are the metadata associated.
  • YAML dictionary where keys are the elements of the lookup and the values are the metadata associated.
  • OTX JSON Pulse.
  • MISP JSON Feed.
