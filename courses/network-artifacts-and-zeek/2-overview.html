In order to demonstrate the power of Artifact Collection this section is going to walk through an example of automating netowrk capture and processing of PCAP files using the the Zeek Network Monitoring Tool.

At a high level, these will be the steps we'll take:

  • Automate network capture on a Linux endpoint.
  • Automate the processing through Zeek.
  • Create D&R rules for the Zeek logs.
