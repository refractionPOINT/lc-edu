The goal of this portion of the course is to create a DR rule to detect the MITRE ATT&CK framework for Control Panel Items execution, as well as validate, test and publish the rule.

Setup and Requirements

To complete this section of the course you will need access to a Linux host (or MacOS terminal with brew installed). Along with this you will also need owner access to an organization on LimaCharlie. If you do not have an organization already you can create one - this code lab is completely compatible with the free tier provided to all LimaCharlie users.

Services Used

This code lab will use the Detection & Response service as well as the Replay service to validate the rule prior to pushing it to production. Ensure the Replay service is turned on in your LimaCharlie organization by going to the Add-Ons -> Services section in the web app and checking the toggle next to "replay".
