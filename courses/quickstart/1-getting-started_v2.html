LimaCharlie is a platform with vast capabilities that enables the user to build any kind of security pipeline or product they can think of. Having said that, the majority of LimaCharlie users make use of common and established patterns.

This guide is designed to get people new to the platform up and running with some solid coverage. It is then up to the user if they want to grow their deployment in complexity or add niche capabilities.

This course will walk through the following:

  • Creating an Organization and installing the Agent
  • Enabling Sigma rule coverage + False Positives
  • Set up an integration to automatically leverage VirusTotal's free tier.
  • Set up an SMTP notification to get notified as soon as a detection takes place.
