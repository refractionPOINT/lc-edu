The LimaCharlie web application is able to be white labelled.

The white label functionality is helpful for MSSPs when they want to exclusively show their branding. A custom white label allows you to use your own custom sub-domain and to have your logo be shown in place of the LimaCharlie logo.

White labelling is a paid add-on at $0.20 / sensor / month.

Instructions on how to make a whitelabel installer for Windows using an MSI wrapper applicaiton can be found here.
